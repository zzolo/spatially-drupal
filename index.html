Tile Serving

Ideal stack for large amounts of data (and good looking maps).

  • Input (or import) geospatial into Drupal.
  • Indexed with PostGIS.
  • Utilize Tilemill to create awesome map styles.
  • Use those map styles with TileStache to server tilesets.
  • Display your maps in Drupal with OpenLayer or Leaflet.
